How to use

  1. Clone or use the terminal template repository as a template.
  2. Open this generator page in a browser.
  3. Fill in your information and click Generate config.
  4. Save the result as terminal-config.js in the template root.
  5. Open index.html of the template in a browser and enjoy your terminal homepage.
